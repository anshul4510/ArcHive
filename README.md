# ArcHive | Global Architecture Repository

> "Where Structure Meets Vision"

ArcHive is a premium digital repository for architectural innovation, functioning as a "GitHub for architectures." It is a collective vault designed to preserve, index, and showcase visionary structural designs—from contemporary brutalism to organic modernism.

---

## 🎨 Design System

### Color Palette
- **Primary Background:** `#F5F3EF` (Warm off-white canvas)
- **Dark Background:** `#0E0E0C` (Near-black sections)
- **Primary Accent:** `#C8A96A` (Gold for highlights and indicators)
- **Secondary Accent:** `#A8894A` (Hover state gold)
- **Text:** `#1A1A1A` (Body) / `#6B6860` (Muted/Editorial)
- **Borders:** `rgba(200, 169, 106, 0.2)` (Subtle gold)

### Typography
- **Display (H1-H2):** `Cinzel` (Monumental, Structural)
- **Subheadings (H3-H4):** `Cormorant Garamond` (Elegant Editorial)
- **Body & UI:** `DM Sans` (Clean Modern)
- **Technical Labels:** `Space Mono` (Data-driven)

---

## 🚀 Technical Stack
- **Framework:** React 19 (Vite)
- **Styling:** Tailwind CSS 3
- **Animations:** Framer Motion (Scroll parallax, theme detection, custom cursor)
- **Icons:** Lucide React

---

## ✨ Key Features
- **Intelligent Navbar:** Uses a custom `IntersectionObserver` to detect background brightness and automatically transition between light and dark themes for perfect visibility.
- **Collective Vault (About):** Reimagined the "Who We Are" section as a global index of 50k+ blueprints and architectural innovations.
- **Hex Motif System:** Integrated "Hive" patterns used in backgrounds, watermarks, and icons to reinforce the repository concept.
- **Monumental Hero:** Features a cinematic wide-angle architectural sweep with a subtle lightened left-side mask for text legibility.
- **Custom Cursor:** A gold hexagonal outline tracking mouse movement (Desktop).

---

## 🛠️ How to Run
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`
